Divisi Web Design
Web Design meliputi berbagai keterampilan yang berbeda dan disiplin ilmu dalam produksi dan pemeliharaan website. Berbagai bidang Web Desain termasuk desain grafis web, desain antarmuka(UI), authoring, termasuk kode standar dan perangkat lunak berpemilik, desain user experience(UX), dan optimasi mesin pencari(SEO).
